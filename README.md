# 🏛️ Umbrella University: The Digital Frontier

Umbrella University is an elite, high-performance digital campus portal designed to redefine the academic web experience. This platform replaces cluttered legacy systems with a streamlined, **"Navy & Gold" themed interface** that balances institutional prestige with modern functional speed.

---

## 🗺️ Information Architecture

The portal is built on a deep, modular hierarchy designed for high scalability:

* 🏛️ **Departments Hub**
  A dynamic directory featuring individual sub-pages for Vision, Faculty Profiles, Research Labs, and HOD contacts. Includes a built-in search filter.

* 🎓 **Academics & Admissions**
  Full transparency on curriculum mapping, fee structures, scholarships, and *"Rag-to-Rich"* student success stories.

* 🚀 **Placements**
  Real-time statistics showcasing a 94% placement rate, top recruiters *(Infosys, TCS, Wipro)*, and an alumni directory.

* 🌿 **Campus Life**
  A rich look into university facilities including Hostels, Canteen food reviews, and *"Reel-style"* Club highlights (Coding, Photography, Dance).

* 🗺️ **Explore BMCE**
  An integrated landing micro-site featuring an interactive map, virtual tour, and a **"Compare Courses"** tool.

---

## ✨ Key Technical Features

### 1. 🧠 Smart Keyword Assistant

A lightweight, zero-latency chatbot that handles **15+ student-specific categories** (Admissions, Syllabus, Events) using a custom-built keyword mapping engine — **no external APIs required**.

---

### 2. 📱 Anti-Overlap Navigation

A responsive architecture that utilizes a **CSS-only "Hamburger" drawer logic** to prevent menu collisions on tablets and mobile devices.

---

### 3. 🟡 The "Gold Standard" UI

* ✨ **Scroll Reveal Engine**
  Uses `IntersectionObserver` for smooth, high-end content reveals.

* 📊 **Success Counters**
  Real-time animated counters for university stats.

* 🎨 **Design Tokens**
  Built with CSS Variables for consistent branding:

  * Navy → `#0A0F1E`
  * Gold → `#C9A84C`

---

## 🛠️ Tech Stack

* **Logic** → Vanilla JavaScript (ES6+)

2. Open the project folder

3. Launch the app:
